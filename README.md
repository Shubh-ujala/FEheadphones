# AudioWave – Premium Audio Store

AudioWave is a modern and minimal e-commerce UI for browsing premium audio products including headphones, earbuds, and speakers.  
The project is built with Next.js (App Router), Tailwind CSS, and TypeScript, focusing on clean UI, fast performance, and simple navigation.

---

## Features

### Product Browsing
- Category-based product listing (Headphones, Earbuds, Speakers, Featured)
- Clean card-based layout
- Bestseller label support
- Consistent and minimal UI design

### Product Details
- Individual product detail view
- Image slider with auto-slide
- Back navigation using `router.back()`
- Product features, ratings, reviews, and additional attributes

### Dynamic Navigation
- Red underline for active category
- Heading automatically updates based on page
- Navigation synced using `usePathname()`

### Contact Page
- Compact contact form
- Inputs for Name, Email, and Message
- Company support email displayed
- Back navigation button
- Designed to match the site’s theme
